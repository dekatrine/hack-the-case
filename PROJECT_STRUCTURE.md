# Hack the Case — Рефакторинг

## Структура проекта (было → стало)

```
БЫЛО:
  app.py (845 строк — всё в одном файле)

СТАЛО:
  app.py                  ← Точка входа (минимальный роутер)
  config.py               ← Константы, отрасли, уровни сложности
  utils/
    llm.py                ← Вызов YandexGPT + retry + логирование
    session.py            ← Инициализация session_state + сохранение данных
    logger.py             ← Логирование действий для исследования (курсовая!)
  prompts/
    case_gen.py           ← Промпт генерации кейса
    coach.py              ← Промпты коуча (с контекстом предыдущих этапов!)
    rubric.py             ← Промпт жюри + JSON Schema + repair
  pages/
    start.py              ← Страница генерации кейса
    solve.py              ← Страница решения по этапам
    evaluate.py           ← Страница оценки (с radar-chart!)
  styles.py               ← CSS стили
```

## Ключевые улучшения

1. **Контекстная память коуча** — коуч видит саммари ответов с предыдущих этапов
2. **Retry-логика API** — 3 попытки с exponential backoff
3. **Промежуточная оценка** — мини-фидбек после каждого этапа
4. **Логирование** — время, действия, для раздела "Результаты" в курсовой
5. **Radar-chart** — визуализация оценки по 10 критериям
6. **Улучшенный stepper** — кликабельная навигация по этапам сверху
