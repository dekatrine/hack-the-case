# Оценка навыка решения кейсов

Для мини-кейсов до и после использовать 6 компетенций.

Шкала для каждой компетенции:

- 0 — компетенция не проявлена;
- 1 — проявлена слабо, ответ фрагментарный или с ошибками;
- 2 — проявлена частично, логика есть, но ответ неполный;
- 3 — проявлена полно и корректно.

## Компетенции

1. Понимание задачи кейса
- клиент
- проблема
- цель
- ограничения
- критерий успеха

2. Структурирование решения
- тип кейса
- логика анализа
- последовательность рассуждения

3. Аналитическая проработка
- сегментация
- анализ рынка/пользователя
- причинно-следственная логика
- опора на данные

4. Качество предложенного решения
- связь инициатив с проблемой
- реалистичность
- механизм влияния на бизнес

5. Метрики и критерии успеха
- наличие измеримых метрик
- связь метрик с решением
- понимание эффекта

6. Финальная рекомендация
- четкий вывод
- аргументация
- риски
- реализация

## Итог

- максимум: 18 баллов;
- считать балл до сервиса;
- считать балл после сервиса;
- считать прирост.
